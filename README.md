# Копия сайта ghk_3999_1

Полная копия сайта https://kirpinev.github.io/ghk_3999_1/ с переходами и блоками.

## Структура проекта

```
.
├── index.html              # Главная HTML страница
├── styles.css              # CSS стили
├── script.js               # JavaScript логика
├── thanks.html             # Страница благодарности
├── images/                 # Папка с изображениями
│   ├── combo-BMCBP6Ec.png
│   ├── smile-DCyyQ7Ks.png
│   └── moai-BXJYLRrx.png
└── README.md               # Этот файл
```

## Запуск

Просто откройте файл `index.html` в браузере двойным кликом или через меню "Открыть с помощью".

**Примечание:** Если сайт не загружается при открытии напрямую (из-за CORS политики браузера), используйте локальный сервер:

```bash
python3 -m http.server 8000
```

Затем откройте в браузере: http://localhost:8000

## Особенности

- Простая архитектура без React/Vite (как в проекте 7112)
- Все пути к ресурсам исправлены для локального запуска
- Сохранены все переходы и блоки оригинального сайта
- Включены все аналитические скрипты (Google Analytics, Yandex Metrika)
- Поддержка адаптивного дизайна
- Работает при открытии напрямую через file://

- **JavaScript** - интерактивность и аналитика
